# מדריכי DS23

אוסף מדריכים אינטראקטיביים בעברית למדעי הנתונים, למידת מכונה ובינה מלאכותית.

## תוכן

| מדריך | ערכים | תחומים |
|------|------|--------|
| [מתמטיקה ל-ML](math_guide_DS23.html) | 113 | אלגברה לינארית, אופטימיזציה, גרפים, תורת האינפורמציה |
| [מילון מושגים סטטיסטי](stats_guide_DS23.html) | 59 | התפלגויות, מבחני השערות, רגרסיה |
| [למידת מכונה ו-AI](ai_ml_guide_DS23.html) | 71 | אלגוריתמי ML, רשתות עצביות, NLP, LLMs |

## מבנה כל מדריך

כל ערך כולל:
- **💡 בקצרה** — הגדרה תמציתית
- **📊 המחשה ויזואלית** — דיאגרמה או גרף
- **🎯 אנלוגיה** — דוגמה מעולם אמיתי
- **🤖 ב-ML** — קונטקסט מעשי
- **✏️ תרגיל ופתרון** — בדיקה עצמית

## תכונות

- 🔍 חיפוש מובנה בכל המדריך
- 📑 תוכן עניינים נשלף
- ⌨️ ניווט במקלדת (חיצים)
